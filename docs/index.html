features

Everything important, quickly.

Restricted-ready

Made for school, office, and shared networks that block normal access.

FPS Unlocker

Remove the 60fps cap with a single toggle. Free: 120fps. Pro: 144, 240, or unlimited.

Discord Rich Presence

Show what you're playing on your Discord profile with game name, time, and join button.

Quick Launch & Favorites

Pin your favorite games for one-click launching via deep links — skip the Roblox app.

Server Region Display

See which datacenter you're connected to in real time with GeoIP lookup.

Auto-Rejoin

Disconnected? SpoofTrap detects it and rejoins the same server automatically.

File Replacement Mods

Swap death sounds, cursors, fonts, app icon, and avatar backgrounds — Bloxstrap-style.

FastFlags & Presets

Curated flag presets plus a full editor for fine-tuning Roblox rendering and behavior.

Multi-Instance

Run multiple Roblox windows at once for alt accounts or testing. Pro only.
