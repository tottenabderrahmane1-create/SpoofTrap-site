FAQ

Common questions.

Do I need Pro to use the bypass?

No. The core bypass works on Free with both App and System proxy modes. Pro only adds power-user features on top.

Does one key work on both macOS and Windows?

Yes. A single Pro license key activates on both platforms. You can deactivate and move it between machines.

Is there a free trial?

No trial, but Free is fully functional — not a crippled demo. Most users never need Pro.

How do I get a Pro key?

Join the Discord server and check the pinned messages or ask in the support channel for license details.

Can I downgrade from Pro?

Yes. Deactivate your license in Settings and the app reverts to Free. Pro features lock but nothing is deleted.

Will Free features stay free?

Yes. Core features (bypass, basic presets, FastFlags toggle, log, mods) will always be free. We don't paywall things retroactively.
